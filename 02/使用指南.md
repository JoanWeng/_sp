# EmoLang 語法與 Emoji 用法指南

EmoLang 是一款結合了 **C 語言結構**與 **Python 動態特性**的 Emoji 程式語言。
以下是完整的語法規則與功能對照表。

---

## 0. 核心語法規則 (Syntax Rules)

在開始寫 EmoLang 之前，請先記住以下三個黃金法則：

1. **空白分隔 (Space is important)**
   Emoji 與變數名稱、運算子之間 **必須有空白**。
   ⭕ 正確：`📦 🍎 🟰 10`
   ❌ 錯誤：`📦🍎🟰10` (直譯器會把它當成一個超長的名字)
2. **換行即結束 (No Semicolons)**
   寫完一行指令後，直接**換行**即可，不需要在結尾加分號 (除非是在 For 迴圈 `🎡` 中使用 `🚧` 分隔條件)。
3. **區塊包裝 (Block Scoping)**
   任何需要包住多行程式碼的地方 (例如 If、迴圈、結構體、函數)，都必須以 `👇` 開始，並以 `👆` 結束。縮排只是為了方便閱讀，真正決定範圍的是手指符號！

## 1. 變數與資料型態 (Variables & Types)

EmoLang 支援動態型別，變數可以隨時改變型態，但你可以用不同的 Emoji 來強調變數的初始用途：

| Emoji | 概念 | 傳統對應 | 語法範例 | 說明 |
| :---: | :--- | :--- | :--- | :--- |
| 📦 | 變數 (通用) | `var` | `📦 a 🟰 10` | 萬用箱子，可裝任何資料。 |
| 🔢 | 整數變數 | `int` | `🔢 b 🟰 20` | 視覺上強調用來儲存整數。 |
| 🎈 | 小數變數 | `float` | `🎈 c 🟰 3.14` | 視覺上強調用來儲存小數。 |
| 📝 | 字串變數 | `str` | `📝 d 🟰 "哈囉"` | 視覺上強調用來儲存文字。支援透過 `➕` 串接文字！ |
| 🚦 | 布林變數 | `bool` | `🚦 e 🟰 🟢` | 儲存真偽值 (`🟢` 代表 True, `🔴` 代表 False)。 |
| 🟰 | 賦值 | `=` | `a 🟰 100` | 將右邊的數值存入左邊的變數中。 |

## 2. 輸入與輸出 (I/O)

| Emoji | 概念 | 傳統對應 | 語法範例 | 說明 |
| :---: | :--- | :--- | :--- | :--- |
| 📢 | 輸出/印出 | `print()` | `📢 "你好"` | 將變數或字串印到終端機螢幕上。 |
| 📥 | 輸入 | `input()` | `📦 x 🟰 📥` | 程式暫停，等待使用者輸入，引擎會自動判斷輸入的是數字還是文字。 |

## 3. 數學與邏輯運算 (Math & Logic)

| Emoji/符號 | 概念 | 語法範例 | 說明 |
| :---: | :--- | :--- | :--- |
| ➕ | 加法/串接 | `a ➕ b` | 數字相加，或將兩段文字合併。 |
| ➖ | 減法 | `a ➖ b` | 數字相減。 |
| ✖️ | 乘法 | `a ✖️ b` | 數字相乘。 |
| ➗ | 除法 | `a ➗ b` | 數字相除。 |
| ✂️ | 取餘數 | `a ✂️ 2` | 除法取餘數（常用於判斷奇偶數）。 |
| `()` | 優先運算 | `(a ➕ 1) ✖️ 2`| 小括號內的運算式優先計算。 |
| 🤝 | 等於 | `a 🤝 10` | 判斷兩邊是否相等。 |
| 📈 | 大於 | `a 📈 5` | 判斷左邊是否大於右邊。 |
| 📉 | 小於 | `a 📉 5` | 判斷左邊是否小於右邊。 |
| 🔗 | 邏輯 AND | `a 🔗 b` | 左右兩邊都為真時才成立。支援短路求值。 |
| 🔀 | 邏輯 OR | `a 🔀 b` | 左右兩邊只要有一個為真即成立。支援短路求值。 |
| 🙅 | 邏輯 NOT | `🙅 a` | 放在條件前面，將真偽值反轉 (True 變 False)。 |

## 4. 條件與迴圈控制 (Control Flow)

| Emoji | 概念 | 傳統對應 | 語法範例 |
| :---: | :--- | :--- | :--- |
| 🤔 | 如果 (If) | `if` | `🤔 🍎 📈 5 👇` |
| 🤷 | 否則 (Else) | `else` | `👆 🤷 👇` |
| 🤷🤔 | 否則如果 | `elif` | `👆 🤷 🤔 🍎 🤝 5 👇` |
| 🔁 | While 迴圈 | `while` | `🔁 🍎 📉 10 👇` |
| 🎡 | For 迴圈 | `for` | `🎡 🔢 i 🟰 0 🚧 i 📉 5 🚧 i 🟰 i ➕ 1 👇` |
| 🚧 | 條件分隔 | `;` | 專門用來分隔 For 迴圈中的「初始、條件、步進動作」。 |

## 5. 函數 (Functions)

EmoLang 具備完整的區域變數 (Local Scope) 隔離能力。

| Emoji/符號 | 概念 | 傳統對應 | 語法範例 | 說明 |
| :---: | :--- | :--- | :--- | :--- |
| 🛠️ | 宣告函數 | `def` / `func` | `🛠️ 計算(a, b) 👇` | 定義一個可重複呼叫的功能區塊。 |
| 🔙 | 回傳值 | `return` | `🔙 結果` | 中斷函數執行，並將資料回傳給呼叫者。 |
| `,` | 參數分隔 | `,` | `🛠️ 函數(x, y)` | 用來分隔多個傳入的參數。 |
| `()` | 函數呼叫 | `()` | `📦 r 🟰 計算(1, 2)`| 接在變數名稱後方代表呼叫函數。 |

## 7. 結構體與指標 (Structs & Pointers)

保留了 C 語言強大的底層記憶體操作能力。

| Emoji | 概念 | C 語言對應 | 語法範例 | 說明 |
| :---: | :--- | :--- | :--- | :--- |
| 🏗️ | 定義結構 | `struct` | `🏗️ 點 👇 🔢 x 🔢 y 👆` | 定義一個新的物件藍圖 (必須在最外層宣告)。 |
| 🆕 | 動態配置 | `malloc`/`new`| `📦 p 🟰 🆕 點` | 在記憶體中建立結構體的實體。 |
| ➡️ | 存取欄位 | `->` 或 `.` | `p ➡️ x 🟰 10` | 存取結構體實體內的專屬屬性。 |
| 📍 | 取址 | `&` | `📦 ptr 🟰 📍 a` | 取得一個變數在記憶體中的真實位址。 |
| 🎯 | 解指標 | `*` | `🎯 ptr 🟰 20` | 操作/修改指標目前所指向的記憶體內容。 |
