# EmoLang 語法與 Emoji 用法指南

EmoLang 是一款結合了 **C 語言結構**與 **Python 動態特性**的 Emoji 程式語言。
其底層為 **零相依 (Zero-Dependency)** 架構，完全不依賴 C 標準庫，直接與作業系統溝通。
以下是完整的語法規則與功能對照表。

---

## 0. 核心語法規則 (Syntax Rules)

1. **空白分隔 (Space is important)**
   Emoji 與變數名稱、運算子之間 **必須有空白**。
   ⭕ 正確：`📦 🍎 🟰 10`
   ❌ 錯誤：`📦🍎🟰10`
2. **換行即結束 (No Semicolons)**
   寫完一行指令後，直接**換行**即可，不需要分號 (唯獨在 For 迴圈 `🎡` 中需使用 `🚧` 分隔條件)。
3. **區塊包裝 (Block Scoping)**
   控制流與函數區塊，必須以 `👇` 開始，並以 `👆` 結束。縮排僅為閱讀方便，真正決定範圍的是手指符號。

## 1. 變數與資料型態 (Variables & Types)

EmoLang 支援動態型別。使用不同 Emoji 可強調變數的初始用途：

| Emoji | 概念 | 傳統對應 | 語法範例 | 說明 |
| :---: | :--- | :--- | :--- | :--- |
| 📦 | 變數 (通用) | `var` / `let` | `📦 a 🟰 10` | 萬用變數，可裝載任何資料型態。 |
| 🔢 | 整數變數 | `int` | `🔢 b 🟰 20` | 視覺上強調儲存整數。 |
| 🎈 | 小數變數 | `float` | `🎈 c 🟰 3.14` | 視覺上強調儲存小數。 |
| 📝 | 字串變數 | `str` | `📝 d 🟰 "哈囉"` | 視覺上強調儲存文字。支援透過 `➕` 串接文字。 |
| 🚦 | 布林變數 | `bool` | `🚦 e 🟰 🟢` | 儲存真偽值 (`🟢`=True, `🔴`=False)。 |
| 🟰 | 賦值 | `=` | `a 🟰 100` | 將右側數值存入左側的變數或記憶體中。 |

## 2. 輸入與輸出 (I/O)

| Emoji | 概念 | 傳統對應 | 語法範例 | 說明 |
| :---: | :--- | :--- | :--- | :--- |
| 📢 | 輸出/印出 | `print()` | `📢 "你好"` | 將變數、字串、列表或字典印到終端機上。 |
| 📥 | 輸入 | `input()` | `📦 x 🟰 📥` | 程式暫停等待輸入，引擎會自動判斷輸入型別(字串/數字)。 |

## 3. 數學與邏輯運算 (Math & Logic)

支援**短路求值 (Short-circuiting)** 及標準運算優先級。

| Emoji/符號 | 概念 | 語法範例 | 說明 |
| :---: | :--- | :--- | :--- |
| ➕ | 加法/串接 | `a ➕ b` | 數字相加，或將兩段文字動態合併。 |
| ➖ | 減法 | `a ➖ b` | 數字相減。 |
| ✖️ | 乘法 | `a ✖️ b` | 數字相乘。 |
| ➗ | 除法 | `a ➗ b` | 數字相除。 |
| ✂️ | 取餘數 | `a ✂️ 2` | 數字取餘數（常用於判斷奇偶數）。 |
| `()` | 優先運算 | `(a ➕ 1) ✖️ 2`| 括號內的運算式優先計算。 |
| 🤝 | 等於 | `a 🤝 10` | 判斷兩側是否相等。 |
| 📈 | 大於 | `a 📈 5` | 判斷左側是否大於右側。 |
| 📉 | 小於 | `a 📉 5` | 判斷左側是否小於右側。 |
| 🔗 | 邏輯 AND | `a 🔗 b` | 兩側條件皆成立時回傳 `🟢` (左側為假時觸發短路)。 |
| 🔀 | 邏輯 OR | `a 🔀 b` | 任一側條件成立時回傳 `🟢` (左側為真時觸發短路)。 |
| 🙅 | 邏輯 NOT | `🙅 a` | 反轉布林值 (`🟢` 變 `🔴`)。 |

## 4. 條件與迴圈控制 (Control Flow)

| Emoji | 概念 | 傳統對應 | 語法範例 |
| :---: | :--- | :--- | :--- |
| 🤔 | 如果 (If) | `if` | `🤔 🍎 📈 5 👇` |
| 🤷 | 否則 (Else) | `else` | `👆 🤷 👇` |
| 🤷🤔 | 否則如果 | `elif` | `👆 🤷 🤔 🍎 🤝 5 👇` |
| 🔁 | While 迴圈 | `while` | `🔁 🍎 📉 10 👇` |
| 🎡 | For 迴圈 | `for` | `🎡 🔢 i 🟰 0 🚧 i 📉 5 🚧 i 🟰 i ➕ 1 👇` |
| 🚧 | 條件分隔 | `;` | 專用分隔符，用於 For 迴圈內的「初始 🚧 條件 🚧 步進」。 |

## 5. 函數 (Functions)

支援區域變數隔離 (Local Scope) 與獨立呼叫堆疊。

| Emoji/符號 | 概念 | 傳統對應 | 語法範例 | 說明 |
| :---: | :--- | :--- | :--- | :--- |
| 🛠️ | 宣告函數 | `def` / `func` | `🛠️ 計算(a, b) 👇` | 定義可重複呼叫的區塊。 |
| 🔙 | 回傳值 | `return` | `🔙 結果` | 中斷函數，並將資料回傳給呼叫者。 |
| `,` | 參數分隔 | `,` | `🛠️ 函數(x, y)` | 分隔多個傳入參數。 |
| `()` | 函數呼叫 | `()` | `📦 r 🟰 計算(1)` | 接於變數後方代表函數呼叫。 |

## 6. 動態列表與字典 (Lists & Dictionaries)

| Emoji | 概念 | 傳統對應 | 語法範例 | 說明 |
| :---: | :--- | :--- | :--- | :--- |
| 📋 | 動態列表 | `[]` / `list` | `📦 a 🟰 🆕 📋` | 建立可動態增減的資料串列。 |
| 📖 | 字典 | `{}` / `dict` | `📦 b 🟰 🆕 📖` | 建立以字串為鍵 (Key) 的雜湊表。 |

| 📌 | 索引 / 鍵 | `a[0]`/`b["k"]`| `b 📌 "名稱"` | 共用存取符號。用於讀取列表位置或字典字串鍵值。 |

## 7. 結構體與指標 (Structs & Pointers)

保留系統級別的記憶體操作能力。

| Emoji | 概念 | C 語言對應 | 語法範例 | 說明 |
| :---: | :--- | :--- | :--- | :--- |
| 🏗️ | 定義結構 | `struct` | `🏗️ 點 👇 🔢 x 🔢 y 👆` | 定義全新的物件藍圖 (需於全域宣告)。 |
| 🆕 | 動態配置 | `malloc`/`new`| `📦 p 🟰 🆕 點` | 在記憶體中建立實體或動態容器。 |
| ➡️ | 存取欄位 | `->` 或 `.` | `p ➡️ x 🟰 10` | 存取結構體實體內的專屬屬性。 |
| 📍 | 取址 | `&` | `📦 ptr 🟰 📍 a` | 取得變數的記憶體絕對位址。 |
| 🎯 | 解指標 | `*` | `🎯 ptr 🟰 20` | 修改/讀取指標目前指向的記憶體內容。 |
