# EmoLang 指令總覽

以下列出 EmoLang 所有支援的指令和語法。

---

## 變數宣告與賦值

| Emoji | 指令 | 範例 | 說明 |
|-------|------|------|------|
| 📦 🔢 🎈 📝 🚦 | LET | `📝 x 🟰 10` | 宣告變數 |
| 🟰 | ASSIGN | `x 🟰 20` | 賦值運算 |

**範例：**
```
📝 x 🟰 10
📝 name 🟰 "Hello"
```

---

## 輸出

| Emoji | 指令 | 範例 | 說明 |
|-------|------|------|------|
| 📢 | PRINT | `📢 x` | 輸出值 |

**範例：**
```
📝 x 🟰 10
📢 x
📢 "Hello World"
```

---

## 輸入

| Emoji | 指令 | 範例 | 說明 |
|-------|------|------|------|
| 📥 | INPUT | `📝 x 🟰 📥` | 讀取使用者輸入 |

**範例：**
```
📝 x 🟰 📥
📢 x
```

---

## 條件判斷

| Emoji | 指令 | 範例 | 說明 |
|-------|------|------|------|
| 🤔 | IF | `🤔 條件 👇 ... 👆` | 條件判斷 |
| 🤷 | ELSE | `🤔 ... 👇 ... 👆 🤷 👇 ... 👆` | 否則 |

**區塊標記：**
| Emoji | 說明 |
|-------|------|
| 👇 | 區塊開始 |
| 👆 | 區塊結束 |

**範例：**
```
📝 x 🟰 10
🤔 x 📈 5 👇
  📢 "大於5"
👆
🤷 👇
  📢 "不大於5"
👆
```

## 迴圈

### While 迴圈
| Emoji | 指令 | 說明 |
|-------|------|------|
| 🔁 | WHILE | `🔁 條件 👇 ... 👆` |

**範例：**
```
📝 i 🟰 0
🔁 i 📉 5 👇
  📢 i
  📝 i ➕ 1
👆
```

### For 迴圈
| Emoji | 指令 | 說明 |
|-------|------|------|
| 🎡 | FOR | `🎡 起始; 條件; 遞增 👇 ... 👆` |
| 🚧 | SEP | `🚧` (分隔符) |

**範例：**
```
🎡 i 🟰 0 🚧 i 📉 5 🚧 i ➕ 1 👇
  📢 i
👆
```

## 運算子

### 算術運算
| Emoji | 指令 | 說明 |
|-------|------|------|
| ➕ | PLUS | 加法 |
| ➖ | MINUS | 減法 |
| ✖️ | MUL | 乘法 |
| ➗ | DIV | 除法 (整數除法) |
| ✂️ | MOD | 取餘數 |

### 比較運算
| Emoji | 指令 | 說明 |
|-------|------|------|
| 🤝 | EQ | 等於 |
| 📈 | GT | 大於 |
| 📉 | LT | 小於 |

### 邏輯運算
| Emoji | 指令 | 說明 |
|-------|------|------|
| 🔗 | AND | 邏輯 AND (短路求值) |
| 🔀 | OR | 邏輯 OR (短路求值) |
| 🙅 | NOT | 邏輯 NOT |

**範例：**
```
📝 a 🟰 🟢
📝 b 🟰 🔴
📢 a 🔗 b   # 輸出: 0 (false)
📢 a 🔀 b   # 輸出: 1 (true)
📢 🙅 a    # 輸出: 0 (false)
📢 🙅 b    # 輸出: 1 (true)
```

## 列表 (List)

### 建立與操作
| Emoji | 指令 | 說明 |
|-------|------|------|
| 📋 | LIST | `🆕 📋` (建立空白列表) |
| 🛒 | APPEND | `list 🛒 值` (追加元素) |
| 📏 | LEN | `📏 list` (長度) |
| 📌 | INDEX | `list📌i` (索引存取) |

**範例：**
```
📦 items 🟰 🆕 📋
items 🛒 "蘋果"
items 🛒 "香蕉"
📢 items              # 輸出: ["蘋果", "香蕉"]
📢 📏 items           # 輸出: 2
items📌0 🟰 "橘子"
📢 items📌0           # 輸出: 橘子
```

## 字典 (Dictionary)

### 建立與操作
| Emoji | 指令 | 說明 |
|-------|------|------|
| 📖 | DICT | `🆕 📖` (建立空白字典) |
| 📌 | INDEX | `dict📌"key"` (鍵值存取) |
| 📏 | LEN | `📏 dict` (長度) |

**範例：**
```
📦 player 🟰 🆕 📖
player📌"name" 🟰 "勇者"
player📌"hp" 🟰 100
📢 player            # 輸出: {"name": "勇者", "hp": 100}
📢 player📌"name"    # 輸出: 勇者
📢 📏 player         # 輸出: 2
```

## 完整範例

### 費波那契數列
```
🛠️ fib n 👇
  🤔 n 📉 2 👇
    🔙 1
  👆
  🔙 fib n ➖ 1 ➕ fib n ➖ 2
👆

📢 fib 10
```

### 計算階乘
```
📝 n 🟰 5
📝 result 🟰 1

🔁 n 📉 1 👇
  📝 result 🟰 result ✖️ n
  📝 n ➖ 1
👆

📢 result
```

## 優先順序

由低到高：
1. `🤔` 條件判斷
2. `🔀` OR
3. `🔗` AND
4. `🤝` `📈` `📉` 比較運算
5. `➕` `➖` 加減
6. `✖️` `➗` `✂️` 乘除餘
7. `🙅` NOT
8. `📍` `🎯` 指標運算
9. `.` `📌` 後綴運算

---

## 語法規則

- 指令之間需有空格分隔
- 區塊必須以 `👇` 開始、`👆` 結束
- 函數參數以逗號分隔
- 字串必須用雙引號包覆 `"..."`
- 註解：不支援
